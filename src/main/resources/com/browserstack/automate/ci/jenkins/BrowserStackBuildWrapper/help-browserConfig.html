Sets the BROWSERSTACK_BROWSERS environment variable to contain a list of JSON encoded browser objects.

    e.g. BROWSERSTACK_BROWSERS='[
        {
            "device": "iPhone 6S",
            "browser": "iphone",
            "browser_version": null,
            "os": "ios",
            "os_version": "9.1"
        },
        {
            "browser": "chrome",
            "browser_version": "48.0",
            "os": "Windows",
            "os_version": "10",
            "device": null
        }
    ]'
